Описание проекта<br />
Данный проект направлен на разработку сайта-портала пользователя для заказа каршеринговых машин, где будет возможность выбрать машину по марке, году и цвету, забрать и оплатить заказ в одном из отделений компании, полный список которых также будет представлен на сайте, или выбрать услугу "доставки машины" по указанному клиенту адресу. Также, будет повзможность оплаты по qr-коду: сгенерированный код на оплату можно будет отсканировать через камеру телефона в приложении банка клиента. Qr-код предоставляется как в отделении комапании, так и водителем, передавшим на указанном месте машину клиенту. Также, на сайте клиентам будет доступен круглосуточный чат поддержки.<br />
